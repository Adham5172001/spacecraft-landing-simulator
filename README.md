A neural network-based simulator for optimising spacecraft landing trajectories under variable environmental conditions. The model learns to predict optimal thrust sequences to achieve soft landings from arbitrary initial conditions.

## Problem Statement

Spacecraft landing is a classic control problem: given an initial position, velocity, and fuel state, determine the optimal thrust sequence to achieve a soft landing (near-zero velocity at touchdown) while minimising fuel consumption.

This project approaches it as a **supervised regression problem**, training a deep neural network on simulated optimal trajectories generated by a physics-based solver.

## Model Architecture

```
Input: [altitude, velocity_x, velocity_y, fuel_remaining, wind_speed, wind_direction]
         │
    Dense(128, ReLU)
         │
    BatchNorm + Dropout(0.2)
         │
    Dense(64, ReLU)
         │
    Dense(32, ReLU)
         │
Output: [thrust_x, thrust_y, thrust_magnitude]  ← continuous control signals
```

## Usage

```python
from simulator import SpacecraftEnv
from model import LandingController

env = SpacecraftEnv(wind_enabled=True)
controller = LandingController.load("models/best_model.h5")

state = env.reset()
done = False
while not done:
    action = controller.predict(state)
    state, reward, done, info = env.step(action)

print(f"Landing result: {info['outcome']}")
print(f"Touchdown velocity: {info['touchdown_velocity']:.2f} m/s")
```
